NuriCheat

This Android App is able to recognize a Nurikabe grid from an image. Then it solves the puzzle.
OpenCV for Java is used to isolate each of the cells on the grid image. Afterwards, each number contained in a cell is classified by a MNST trained neural network.
In the final step, the app solves the puzzle and shows the correct solution.

Fruit Inspection

This particular project, developed in a two people team for my Computer Vision course, aimed to detect defects in various images of fruit.
It mainly uses OpenCV and Jupyter Notebook to naturally explain its processes and logic.

Keyforge Deck Evaluation

The goal of this project was to create and train a Neural Network model that could grade decks from the KeyForged card game.
Each deck was already scored based on its features and cards by a fan-made database that used a grading system called SAS. The model would then be trained from datasets created with the mentioned database using Pandas and TensorFlow.
